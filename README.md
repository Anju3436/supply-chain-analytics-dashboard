

# 📊 Supply Chain Analytics Dashboard
🚀 Power BI Project | Data Analytics|Business Intelligence.

Interactive Supply Chain Analytics Dashboard built in Power BI to uncover business insights and optimize logistics performance.

# 📌 Project Overview
This project showcases a Supply Chain Analytics Dashboard built using Power BI to analyze and visualize key business metrics.
The dashboard helps in understanding revenue trends, cost distribution, logistics performance, and customer behavior to support data-driven decision-making.

# 🎯 Objectives
Analyze overall revenue, cost, and profit.

Evaluate shipping cost by transportation modes.

Identify top-performing product categories.

Understand customer segmentation.

Compare location-wise logistics performance.
